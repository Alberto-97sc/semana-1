✏️ Comandos de la terminal que aprendi

Comando Descripcion

cd Cambiar de directorio
del Borra el archivo o carpeta
md Mueve un archivo o directorio a otra ubicacion
code . Abre la carpeta donde te encuentras en VSCode

Alias Descripcion
